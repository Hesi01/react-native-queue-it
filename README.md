# react-native-queue-it

A React Native Queue-It Wrapper

## Installation

```sh
npm install react-native-queue-it
```

## Usage

```js
import QueueIt from "react-native-queue-it";

// ...

const result = await QueueIt.multiply(3, 7);
```
